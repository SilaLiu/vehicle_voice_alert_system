# Vehicle Voice Alert System

## 確認環境
- Ubuntu20.04
- Python3.5
- ros2 galactic

## setup

### Install Autoware
下記参照
https://github.com/tier4/AutowareArchitectureProposal.proj

### setup FOA
```
cd vehicle_voice_alert_ystem
source {AUTOWARE_PATH}/install/setup.bash
bash setup.sh
```


## start
```
source {AUTOWARE_PATH}/install/setup.bash
bash start.sh
```

## rebuild
```
cd vehicle_voice_alert_ystem
colcon build
```
